The Decimatorinator V1.0

Installation instructions:

Open Blender > Edit > Preferences > Add-ons > Install... 

**_____________________________________________________________________________**

Use Instructions:

- Place all models to be decimated in a single folder -> Select as input folder

- Select an output folder

- Select decimation ratio

- Select decimation type

- Decimate

**_____________________________________________________________________________**


Addition in the future: Passthrough based on polycount
